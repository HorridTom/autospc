# Example Time Series 2b

A third simulated example time series used in illustrating features of
the Stable Shift Algorithm. Identical to example_series_2a except at
seven time points, all occurring at or after x = 27.

## Usage

``` r
example_series_2b
```

## Format

A data frame with 43 rows and 2 columns

- x:

  Consecutive integer time

- y:

  Integer time series

## Source

Simulated data
