# Example Time Series 2c

A fourth simulated example time series used in illustrating features of
the Stable Shift Algorithm. The first 43 rows are identical to
example_series_2a except at six time points, all occurring at or after x
= 38.

## Usage

``` r
example_series_2c
```

## Format

A data frame with 47 rows and 2 columns

- x:

  Consecutive integer time

- y:

  Integer time series

## Source

Simulated data
